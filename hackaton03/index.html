Hackaton 03

Hacer un algoritmo en JavaScript que lea un número por el teclado y determinar si tiene tres dígitos. 

btn_hackaton03.addEventListener('click', () => {

    const valor = Number(inputEjercicio1.value);

    if(isNaN(valor)) {
        return alert('No es numero');
    }

    if(valor >= 100 && valor <= 999) {
        return alert('El numero tiene 3 cifras.');
    }

});
